# Z-Defender-Landing-Page
Developed a responsive landing page implementing modern frontend techniques including parallax effects, CSS animations, and DOM-based interactivity. Ensured cross-device compatibility using media queries and optimized UI for performance and usability.
## 🚀 Features
- Responsive design (mobile + desktop)
- Parallax scrolling effect
- Floating product animation
- Interactive UI elements
- Clean and modern layout
